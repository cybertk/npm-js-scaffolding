## Usage

[mffr][] is required.

    git clone https://github.com/cybertk/npm-js-scaffolding.git
    cd npm-js-scaffolding
    ./bootstrap.sh <your_package_name> <dest_dir>

[mffr]: https://github.com/cybertk/mffr
